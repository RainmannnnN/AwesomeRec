## Curated Paper Lists
* [Awesome-Generative-Recommendation](https://github.com/uestc-huangyw/Awesome-Generative-Recommendation) - 专注于工业级应用的**生成式推荐**论文合集。
* [Multitask-Recommendation-Library](https://github.com/easezyc/Multitask-Recommendation-Library) - **MTReclib**: 提供多任务推荐模型（MTL）的 PyTorch 实现及常用数据集。
* [Awesome-Deep-Learning-Papers-for-Search-Recommendation-Advertising](https://github.com/guyulongcs/Awesome-Deep-Learning-Papers-for-Search-Recommendation-Advertising) - 涵盖搜索、推荐、广告（搜广推）全链路的深度学习论文集，包含召回、排序、LLM 等。

## 开源框架
* [RecBole (伯乐)](https://github.com/RUCAIBox/RecBole) - 全方位、全面、高效的推荐系统集成库，支持多种底层框架。
* [DeepCTR](https://github.com/shenweichen/DeepCTR) - 易于使用的深度学习点击率（CTR）预测模型封装，支持 TF/PyTorch。
* [FuxiCTR](https://github.com/reczoo/FuxiCTR) - 专注于可复现性的 CTR 预估实验框架。
* [mlgb](https://github.com/UlionTse/mlgb) - 包含 50+ 推荐系统深度模型，支持双框架实现的轻量级库。
* [RecZoo](https://github.com/reczoo/RecZoo) - 推荐系统模型动物园，收集了各类 SOTA 模型的简洁实现。
* [ColdRec](https://github.com/YuanchenBei/ColdRec) - 专门针对**冷启动**推荐问题的开源工具箱。

## 开源项目
* [MiniOneRec](https://github.com/AkaliKong/MiniOneRec) - 对 OneRec 框架的极简复现，适合学习底层逻辑。
* [Recommand-Algorithme](https://github.com/HuichuanLI/Recommand-Algorithme) - 总结了工业界常用的推荐算法及其具体实现。
* [新闻推荐系统](https://github.com/LogicJake/tianchi-news-recommendation) - 天池新闻推荐大赛 TOP2 方案，包含完整的特征工程与建模流程。

## 开源课程
* [动手学大模型 (Dive into LLMs)](https://github.com/Lordog/dive-into-llms) - 紧跟 LLM+Rec 趋势的编程实践教程。
* [llm-course](https://github.com/mlabonne/llm-course) - 系统的 LLM 学习路线图，包含 Colab 笔记本实践。
* [王树森推荐系统公开课](https://github.com/wangshusen/RecommenderSystem) - 工业界广受好评的推荐系统入门与进阶视频课讲义。
* [FunRec](https://github.com/datawhalechina/fun-rec) - Datawhale 开源的推荐系统入门教程，适合零基础同学。

## 数据集
* [RecSysDatasets](https://github.com/RUCAIBox/RecSysDatasets) - 推荐系统公开数据集的导航仓库，再也不用担心找不到数据。
